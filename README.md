# Privacy Policy for Ethyf

Ethyf does not collect personally identifiable information from users. We use local device storage to manage family finance data securely. No data is transmitted to external servers.

## Data Usage
- All financial records are stored locally on your device.
- We do not access, share, or sell any user data.
- No third-party SDKs are used for tracking or analytics.

## Children’s Privacy
Ethyf is not intended for children under 13. We comply with Google Play’s User Data Policy.
